主語・述語ゲーム（Subject-Predicate Matcher）
言葉の組み合わせを楽しく学べる、シンプルなPythonプログラムです。 正解するとAI先生から心のこもった感謝のメッセージが届きます。

📝 アプリの概要
バラバラになった「主語」と「述語」を正しく組み合わせて、正しい文章を作るクイズゲームです。 楽しみながら日本語の構成を学ぶことができます。
